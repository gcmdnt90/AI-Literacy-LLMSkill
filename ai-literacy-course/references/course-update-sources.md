# Course Update Sources

Use this file when asked to update the course or when the tracker is stale.

## Update Policy

Search or browse when available. If search is unavailable, state that the course cannot be verified as current and rely only on the existing standards map.

Update the course when a source changes one of these:

- A core AI literacy framework or competency model.
- Validated assessment constructs or short forms.
- Pedagogical guidance for AI in teaching and learning.
- Tool behavior that affects lessons, examples, risks, or verification workflows.
- Legal or policy requirements in the learner's jurisdiction.

Do not update the course for ordinary product news unless it changes what the learner should know or do.

## Priority Sources

- Stanford Teaching Commons AI Teaching Guide: https://teachingcommons.stanford.edu/teaching-guides/artificial-intelligence-teaching-guide
- Stanford Understanding AI Literacy: https://teachingcommons.stanford.edu/teaching-guides/artificial-intelligence-teaching-guide/understanding-ai-literacy
- MAILS original paper: https://doi.org/10.1016/j.chbah.2023.100014
- MAILS short version / further validation: https://doi.org/10.1016/j.heliyon.2024.e39686
- GAAIS original validation: search for Schepman and Rodway, General Attitudes towards Artificial Intelligence Scale.
- GAAIS-IT: https://doi.org/10.1186/s40359-025-02935-2
- Generative AI Literacy: Twelve Defining Competencies: https://doi.org/10.1145/3685680
- EU AI Act official text when relevant: https://eur-lex.europa.eu/
- Official documentation for specific tools named by the learner.
