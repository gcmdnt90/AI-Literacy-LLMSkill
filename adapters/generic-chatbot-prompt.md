# Generic Chatbot Prompt

Paste this into any chatbot that does not natively support `SKILL.md`.

```text
Use the following markdown skill as your operating procedure for this conversation. You are not merely summarizing it; you should follow it.

Skill: ai-literacy-course
Goal: create, run, update, and track my personal AI literacy course.

Persistence rule: maintain a durable markdown tracker named ai-literacy-program.md. If you cannot create or preserve a file/artifact/project note, tell me before relying on memory and provide exact markdown I can store.

Teaching rule: use embedded diagnosis first, occasional quizzes only when pedagogically useful, and update the tracker with evidence-based observations.

Freshness rule: when web/search is available, check authoritative sources for course updates. If web/search is unavailable, say so and do not claim the course is current.

Now read and follow the attached or pasted SKILL.md.
```
